I generated a single CSV template that gives you:

A minimal, reproducible evaluation matrix (benchmark version pinning, dataset/split, prompt+decoding invariants, parsing/eval script versioning)

A stable single-results schema (so you can compare runs over time)

ChartJS-friendly fields (chart_label, chart_metric, chart_value, chart_series)

Reference fields you can render under the chart (ref_key, ref_text, ref_url, plus per-row source_*)
